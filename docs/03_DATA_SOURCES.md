# 🔌 Mapeamento de Fontes de Dados (Data Sources)

## 1. Yahoo Finance API (`yfinance`)
* [cite_start]**Uso:** Cotações diárias, volumes e histórico de Dividendos/JCP[cite: 47, 48].
* **Limitação:** Proventos passados em moeda brasileira podem sofrer distorções por desdobramentos/grupamentos se não ajustados adequadamente.

## 2. Portal de Dados Abertos CVM
* [cite_start]**Uso:** Demonstrações Financeiras Padronizadas (DFP) e Informações Trimestrais (ITR)[cite: 111].
* [cite_start]**Frequência:** Atualização em lote (Sexta-feira) para checar novos lançamentos corporativos[cite: 114].

## 3. B3 / Sistema Fundos.net
* [cite_start]**Uso:** Captura dos Relatórios Gerenciais Mensais dos fundos imobiliários GARE11 e HSLG11[cite: 112, 116].
